Interactive BOINC web interface for known Docker containers

Submit jobs via boinc2docker.



Choose a known Docker image and submit a command, no quotes necessary.
i.e.:
"cd /path/to/somewhere; python a.py" instead of
"bin/boinc2docker_create_work.py python:alpine /bin/bash -c "python -c print('Hello BOINC')" ""

Results are moved into the server by default. Only use single quotes (' ').
